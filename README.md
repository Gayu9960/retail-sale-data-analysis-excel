# 📊 Retail-sales-data-analysis-excel (Excel Project)

A simple **sales data analysis project** performed using **Microsoft Excel**.
This project analyzes Vrinda Store sales data to identify trends, top-performing products, and overall business insights.

---

## 🎯 Project Objective

The objective of this project is to analyze store sales data and generate useful insights that can help understand product performance and sales trends.

---

## 🚀 Technologies Used

* Microsoft Excel
* Data Cleaning
* Pivot Tables
* Charts and Graphs

---

## ✨ Features

* Cleaned and organized raw sales data
* Created **Pivot Tables** to summarize data
* Analyzed **monthly sales trends**
* Identified **top selling products**
* Visualized insights using **charts and graphs**
