# retail-sale-data-analysis-excel
This project analyzes sales data of Vrinda Store using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Data Cleaning
- Pivot Tables
- Charts
- Data Analysis

## Project Objective
To analyze sales performance and identify trends in product sales.

## Key Insights
- Identified top selling products
- Analyzed monthly sales trends
- Compared sales across categories
